# Untangling Systems Newsletter

Single-page landing page draft for the Untangling Systems newsletter.

## What’s included

- GitHub Pages-friendly `index.html`
- publication-first, author-backed positioning
- recommended landing page architecture embedded into the page
- first-pass copy draft
- editorial visual design direction
- placeholder subscribe forms ready to swap with ConvertKit embed/action

## Notes

The page is intentionally custom rather than ConvertKit-styled so the newsletter can feel like a real publication with stronger visual identity and better SEO surface area.
